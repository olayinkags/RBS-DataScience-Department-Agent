# RBS-DataScience-Department-AI-Chatbox

### How to run?
1. Create an environment

```bash
# make sure you have conda initialized for PowerShell (run `conda init powershell` once)
conda create -n Chatbox python=3.10 -y
conda activate Chatbox        # use the conda command, not a path to Scripts
```

2. Install requirements

```bash
pip install -r requirements.txt

```
